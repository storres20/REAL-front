## Guía de instalación
---
## Project setup

In the project directory, you can run:

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.

The page will reload if you make edits.
